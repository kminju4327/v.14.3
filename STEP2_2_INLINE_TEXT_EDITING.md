# STEP 2.2 Inline Text Editing

- Removed the Section Editor modal from the final detail-page preview.
- Removed the AI improvement request UI from the section text-edit flow.
- Clicking Edit now switches only that section into inline editing mode.
- Title, body, and item text can be edited directly in the preview.
- The upper-right control changes from Edit to Save / Cancel while editing.
- Save applies the text without regenerating or changing the background image.
- Cancel restores the original text.
- Hero copy supports the same inline-edit flow.
